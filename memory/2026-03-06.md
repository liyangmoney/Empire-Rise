- **UI 面板**: ResourcePanel, BuildingPanel, ArmyPanel (基础版)
- **Socket 集成**: 完整的事件系统支持

### 遇到的问题与解决
1. **空白页面问题**
   - 原因: 未启用 DOM 插件
   - 解决: 添加 `dom: { createContainer: true }`

2. **布局超界 + 画面模糊**
   - 解决: 禁用自动缩放 `mode: Phaser.Scale.NONE`
   - 禁用抗锯齿 `antialias: false`
   - 启用像素对齐 `roundPixels: true`

3. **连接成功无反应**
   - 原因: `empire:init` 事件监听时机不对
   - 解决: 在 `create()` 一开始就设置监听器

4. **输入框使用 prompt 弹窗**
   - 改进: 实现直接在 Phaser 内输入
   - 添加闪烁光标、键盘事件处理
   - 支持退格、删除、方向键、Home/End
